面板数据识别策略-固定效应模型

固定效应模型

1. 面板数据

面板数据（panel data）指的是在一段时间内跟踪同一组个体的数据。它既有横截面数据（n个个体），又有时间维度（T个时期）。

面板数据有好的优点，所以我很喜欢面板数据

  1. 可以解决遗漏变量问题。遗漏变量常常是由于不可观测的个体差异或者是异质性（比如说：教育内含个体能力）造成的，如果这种个体差异不随时间而改变（time invariant），那么面板数据提供了解决遗漏变量的方法
  2. 提供更多个体动态行为的信息。面板数据同时有横截面和时间两个维度，比如，卡吕如何区别规模效应与技术进步对企业生产效率的影响。对于截面数据由于没有是时间维度，故没有办法观测到技术进步。而对于单个企业的时间序列数据而言，我们没有办法去区别其生产效率的提高究竟有多少是由于规模的扩大，有多少是由于技术进步造成的
  3. 样本容量更大。面板数据同时又截面维度和时间维度，通常面板数据的样本容量更大，从而可以提高估计的精度

面板数据的估计策略一般有三种：

  • 混合回归
  • 固定效应模型
  • 随机效应模型

2. 混合回归

混合回归指的是所有个体都拥有完全一样的回归方程，即使方程截距项和斜率相同（），将这些相同的个体效应统一记为 可写为：
$$
{y_{it} }=\alpha +x’{it} {\beta}+z’{i}\delta+ \varepsilon_{it} \tag{1}
$$
公式（1）中的 不包含常数项，此时混合回归将所欲哦数据放在一起，像是对待截面数据那样进行 OLS 回归。混合回归的基本假设是不存在任何的个体效应，将样本中的所有数据一视同仁，硬要进行 OLS 回归。

3. 固定效应模型

在面板数据中最常用的识别策略就是固定效应模型了，所以固定效应模型指的是为每位个体估计方程拥有相同的斜率，但可以有不同截距项以捕捉异质性
$$
{y_{it} }= x’{it} {\beta}+z’{i}\delta +{u_i}+ \varepsilon_{it} \tag{2}
$$
其中， 为不随时间变化的个体特征，比如：性别；而 可以随个体及时间而变。扰动项由 两部分组成，称为“复合扰动项”。其中，不可观测的随机变量 是代表个体异质性的截距项，即使“个体效应“。如果 与某个解释变量相关，则进一步称之为”固定效应模型“（Fixed Effects Model,FE），但是在这种情况下的 OLS 估计不一致，解决的方法是将模型转换，消去 后获得一致的估计量。

固定效应模型估计的方法有三种：

  1. 组内估计量（去心处理）
  2. LSDV法（设置虚拟变量）
  3. 一阶差分法（差分处理）
3.1 组内估计量

考虑以下的固定效应模型：
$$
{y_{it} }= x’{it} {\beta}+z’{i}\delta +{u_i}+ \varepsilon_{it} \tag{2}

{\bar y_i} = {\bar x’{it} }\beta + {z’_i}\delta + {u_i} + {\bar \varepsilon _{it} } \tag{3}
$$
其中，${ {\bar y}_i} \equiv \frac{1} {T}\sum\nolimits{t = 1}^T { {y_{it} } }{\bar x’{it} }{\bar \varepsilon _{it} } z{i}u_i$
{y_{it} }-{\bar y_i}=(x_{it}-{\bar x_{i} })’ {\beta} +( \varepsilon_{it}- {\bar \varepsilon {it} } ) \tag{4}
$$
其中，$z’{i}\delta+{u_i}{ {\tilde y}{it} } \equiv {y{it} } - { {\bar y}i} \ , \ { {\tilde x}{it} } \equiv {x_{it} } - { {\bar x}i} \ , \ { {\tilde \varepsilon}{it} } \equiv {\varepsilon_{it} } - { {\bar \varepsilon}i}$
{\tilde y}{it}= \tilde x’{it}\beta+{\tilde \varepsilon _{it} } \tag{5}
$$
上式中已经将个体异质性 消去，只要新扰动项 与新解释变量 ${\tilde x’{it} }\beta\hat{\beta_{FE} }\hat{\beta_{FE} }u_ix_{it}$ 相关，只要使用组内估计量就可以得到一致估计。考虑到可能存在组内自相关，所以应该使用每位个体为聚类的聚类稳健标准误。另外需要强调的是，代表个体异质性变量 $z’{i}\delta\delta\hat{\beta{FE} }$ 无法估计不随时间而变的变量的影响，否则会汇报 ommited，这个也算是固定效应的一个缺点吧。

上述组内估计量的估计过程需要有一个假设：新扰动项 ${\tilde \varepsilon {it} } {\tilde x’{it} }(\varepsilon_{it}- {\bar \varepsilon {it} } ) (x{it}-{\bar x_{i} })iE(\varepsilon_{it} | {x_{i1},x_{i2},{…},x_{iT} })=0$。

3.2 LSDV法

对于式子（2）的个体固定效应 ，传统上将其视为个体 的待估参数，具体地说，可以看成 是个体 地截距项。而针对 位个体的 个不同的截距项，恶意通过在方程（3）中引入 个个体虚拟变量来实现，即是估计以下模型：
$$
{y_{it}} = \alpha + { {x’}{it} }\beta + { {z’}_i}\delta + \sum\nolimits{i = 2}^n { {\gamma _i} } {D_i} + {\varepsilon _{it} } \tag{6}
$$
其中，个体虚拟变量 说明该方程固定的个体是个体 2 ；否则 。然后用 OLS 进行方程参数估计，称为”最小二乘虚拟变量法“（Least Square Dunmmy Variable，LSDV）。因为线性回归与离差形式的额回归在某种意义上是等价的，所以 LSDV 法的估计结果与上述的组内估计量 FE 完全相同。使用 LSDV 的好处是可以得到对个体异质性 的估计，但也存在一些缺点，就是如果 n 很大的时候，则需要在方程中加入很多的虚拟变量，可能会超过 Stata 所允许的变量个数。

3.3 一阶差分法

对于传统的固定效应模型，还可以对原方程（3）两边进行一阶差分，以此来消除个体效应 ，但是同时也把 $z’{i}\delta$
{y{it } } - {y_{i,t - 1} } = ({x_{it} } - {x_{i,t - 1} })’\beta + ({\varepsilon {it} } + {\varepsilon _{i,t - 1} }) \tag{7}
$$
对上面的式子使用 OLS 既可以得到”一阶差分估计量“（First Differencing Estimator，FD），记为 $\hat{\beta{FD} }u_i ({\varepsilon {it} } + {\varepsilon _{i,t - 1} }) ({x{it} } - {x_{i,t - 1} })\hat{\beta_{FD} }\hat{\beta_{FE} }\hat{\beta_{FD} }$ 的优点。

4. 时间固定效应模型

个体固定效应模型解释了不随时间而变但随个体而异得遗漏变量问题。但是现实中还存在不随个体而变但随时间而变的遗漏变量问题；比如：企业经营的宏观经济环境、地区产业的技术进步等等。所以在使用个体固定效应模型的基础上再加入世家固定效应 ：
$$
{y_{it} }= x’{it} {\beta}+z’{i}\delta +{u_i}+ \lambda_t+ \varepsilon_{it} \tag{8}

{y_{it}} = \alpha + { {x’}{it} }\beta + { {z’}_i}\delta + \sum\nolimits{t = 2}^T { {\gamma i} } {D_i} + {\varepsilon _{it} } \tag{9}

{y{it} }= x’{it} {\beta}+z’{i}\delta +{u_i}+ \gamma{t}+ \varepsilon_{it} \tag{10}
$$
上面方程的假定是每个时期的时间效应相等，即每一期都增加 。如果这个假定不太可能成立，则应该在原方程中加入时间虚拟变量。

5. 随机效应模型

考虑以下的随机效应模型：
$$
{y_{it} }= x’{it} {\beta}+z’{i}\delta +{u_i}+ \varepsilon_{it} \tag{11}
$$
其中，个体效应 与解释变量 {} 都是不相关的，所以 OLS 估计出来的结果是一致的。但是由于扰动项是由 组成，并非是球形扰动项，所以估计出来的结果不是最有效率的。

所以说，固定效应模型和随机效应模型的差别在于：遗漏的个体特征变量，究竟是算作解释变量，还是算作随机误差项？其中固定效应模型认为个体特征变量应该算如解释变量中，随机效应模型则将个体特征变量考虑进随机误差项中。也正是如此，固定效应模型中的解释变量可以与个体特征变量相关，但是随机效应中不能。

所以当我们处理面板数据时候，到底是使用固定效应还是随机效应模型是一个根本问题。为此，设立检验原假设 （及随机效应模型为正确模型）。首先不论原假设是否成立，我们通过 FE 估计出来的结果都是一致的，只是当原假设成立的话，RE 的结果估计会比 FE 更有效率，并且估计量会共同收敛于真实的参数值，二者的差距将在大样本下消失，故 ；当原假设不成立，我们就应该使用 FE，且是 RE 在拒绝原假设的情况下估计不一致，故二者估计量差距过大。以二次型度量此距离，豪斯曼检验的统计量为：
$$
(\hat{\beta_{FE} }-\hat{\beta_{RE} })’[ {\widehat {Var( { {\hat \beta }{FE} } - { {\hat \beta }{RE} })} } ]^{ - 1}(\hat{\beta_{FE} }-\hat{\beta_{RE} }) \xrightarrow{d} {\chi ^2}(K) \tag{12}
$$
其中， 为 的维度，即是 中所包含的随时间而变的解释变量个数（ 无法估计不随时间而变得解释变量系数）。如果该统计值大于临界值，则拒绝原假设 。这个检验存在的缺点是，为了计算 ${\widehat {Var( { {\hat \beta }{FE} } - { {\hat \beta }{RE} }) } } H_0\hat{\beta_{RE} }\hat{\beta_{RE} }$ 并非是最有效率的估计量。因为，传统的豪斯曼检验斌不适用异方差的情形，需要使用异方差稳健的豪斯曼检验。

7. 面板模型实例

7.1 面板数据设定的stata命令

设定面板数据的命令为：

xtset panelvar timevar

命令 xtset 告诉 stata 你的数据为面板数据，其中面板个体变量 panelvar 的取值必须要是整数且不重复，相当于将样本中每位个体进行编号；而 timevar 为时间变量。

现实面板数据统计特征的命令包括：

xtdes    //显示面板数据的结构，是否为平衡面板
xtsum    //显示组内、组间与整体的统计指标
xtline varname  //对每个个体分别显示该变量的时间序列图，如果希望将所有的个体的时间序列图叠放在一起，可以加上选择项 overlay
7.2 面板模型案例

本文将采用数据集 lin_1992.dta 为例子，该数据研究家庭联产承包责任制与中国农业增长的经典研究。该省级面板包含中国28个省的1970-1987年有关种植业的数据，被解释变量是“种植业产值对数（ltvfo）”，解释变量包括：耕地面积对数（ltlan），种植业劳动力（ltwlab），机械动力与畜力对数（ltpow），化肥施用量对数（ltfer），截至年底采用家庭联产承包责任制的生产对比重（hrs），农村消费者价格与农村工业投入品价格之比的一阶滞后（mipric1），超额收购价格与农村工业投入品价格之比（giprice），复种指数（mci），非粮食作物占播种面积比重（ngca），时间趋势，province（省），year（年）。

  • 豪斯曼检验

在处理面板数据的时候，使用固定效应还是随机效应模型是一个基本问题，一般情况下，我们会进行豪斯曼检验进行分析，豪斯曼检验的stata命令为

xtreg y x1 x2 x3,fe
est store fe
xtreg y x1 x2 x3,re
est store re
hausman fe re,constant sigmamore

其中，选择项 constant 指的是比较系数估计值时包含常数项；sigmamore 表示统一使用更有效率的那个估计量（随机效应估计量比固定效应估计量更有效率）的方差估计。

xtreg ltvfo ltlan ltwlab ltpow ltfer hrs mipric1 giprice mci ngca,fe
est store fe
xtreg ltvfo ltlan ltwlab ltpow ltfer hrs mipric1 giprice mci ngca,re
est store re
hausman fe re,constant sigmamore

由于 p 值为 0.0000，所以强烈拒绝原假设 ，可以认为使用固定效应模型（FE），而非是随机效应模型（RE）。

传统的豪斯曼检验假定，在原假设成立的情况下，随机效应模型最有效率，这意味着扰动项必须是同方差的，在异方差的情况下是不适用的。因为传统的豪斯曼检验假设球形扰动项，所以在进行固定效应与随机效应的估计时，都是不适用异方差或者是聚类稳健的标准误。所以我们可以通过 xtoverid 来进行稳健的豪斯曼检验，此处的 overid 指的是 overidentification test （过度识别检验）

ssc install xtoverid
quietly xtreg ltvfo ltlan ltwlab ltpow ltfer hrs mipric1 giprice mci ngca,r
xtoverid

Test of overidentifying restrictions: fixed vs random effects
Cross-section time-series model: xtreg re  robust cluster(province)
Sargan-Hansen statistic 225.909  Chi-sq(9)    P-value = 0.0000

上表显示，卡方统计量为 221.225，p 值时0.000，所以可以强烈拒绝随机效应的原假设。
